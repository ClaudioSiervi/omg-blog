## OMG Blog


**Run the project with MacOs** 

Install pipenv
`brew install pipenv`   
  
Start the virtualenv
`pipenv shell`   

Install the dependancies
`pipenv install`

Run:    
`python manage.py runserver 8899`    


Home http://localhost:8899/   
Backoffice http://localhost:8899/admin
